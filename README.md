## Machine Learning-Based Student Health Risk Predictor

## Problem Statement
Students generally ignore their health due to academic pressure and dailylife habits. This project predicts a student's health risk level based on their daily habits. 

## Objective 
To build a simple machine learning model that classify health risk levels.

## Technologies|Techniques  Used 
* Python(IDLE)
* Pandas
* Scikitlearn
* joblib
## Features(input\output) 
* Input:- Sleep, Screen Time, Exercise, Stress
* Output:- Health Risk Level (Low/Medium/High) 
## How to Run(way) .
1. Install required libraries: pip install pandas scikit-learn
2. Run the program: python main.py 

## Model Used 
* Decision Tree Classifier
* random forest classifier
